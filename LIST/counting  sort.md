# 카운팅 정렬 알고리즘

- 평균 수행 시간 : O(n+k)
  
- 최악 수행 시간 : O(n+k)
  
- n : 리스트 개수, k : 정수의 최댓값
  
- n이 k보다 클 경우 : O(n)
  
  ### 조건
  
  - 리스트 내의 모든 원소들은 정수의 범위
    
  - k = O(N) 으로 나타낼 수 있어야 함
    
  
  ### 장점
  
  - 적은 개수의 숫자를 정렬할 때 빠른 속도로 정렬이 가능,
    
  - 안정 정렬
    
  - 원본을 건들지 않는다.
    
  
  ### 단점
  
  - 항목 개수를 저장할 공간, 결과를 저장할 공간 등 추가적인 메모리 필요
